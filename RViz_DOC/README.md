# CONFIGURACIÖN DENTRO DE RVIZ2
Como lanzamos RViz2 sin un archivo de configuración predeterminado, la pantalla estará vacía al principio. 

Para ver al Go2:
* En el panel izquierdo, busca Global Options -> Fixed Frame. Cambia el valor (que probablemente diga map) escribiendo base o trunk (el nombre del eslabón base de tu robot).

  <img width="342" height="155" alt="image" src="https://github.com/user-attachments/assets/9bb9b90e-f41a-4210-9953-abc0a9d04491" />

* Abajo a la izquierda, haz clic en el botón Add.
  Selecciona RobotModel y haz clic en OK.

  <img width="502" height="811" alt="image" src="https://github.com/user-attachments/assets/bea8d8ba-a653-4311-b3ca-8f1d8cab34de" />

* Expande la pestaña de RobotModel que acaba de aparecer en el panel izquierdo y asegúrate de que Description Topic esté configurado en /robot_description.
